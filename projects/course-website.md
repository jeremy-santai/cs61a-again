# CS 61A Course Website

**Official Site:** [https://cs61a.org](https://cs61a.org)

## Overview

CS 61A: Structure and Interpretation of Computer Programs is UC Berkeley's introductory computer science course. It teaches programming fundamentals using Python and Scheme.

## Key Resources on the Site

- **Lecture slides and videos**
- **Lab assignments** — Hands-on practice problems
- **Homework assignments** — Weekly problem sets
- **Projects** — Major assignments (Hog, Cats, Ants, Scheme)
- **Discussion worksheets** — Section practice problems
- **Past exams** — Midterms and finals with solutions

## Course Topics

1. Functions and control
2. Higher-order functions
3. Recursion
4. Data abstraction
5. Trees and linked lists
6. Object-oriented programming
7. Scheme language
8. Interpreters
9. Declarative programming (SQL)
